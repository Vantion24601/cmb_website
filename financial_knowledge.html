MODULE 01

认识金融与银行

金融与我们的日常生活密切相关。掌握基础金融知识，是提高金融素养的重要起点，也是理性参与金融活动的基础前提。

💡

什么是金融

金融的核心作用是促进资金在个人、企业和社会之间合理流动，从而更好地服务生产生活与经济发展。无论是储蓄、支付、贷款，还是财富管理，背后都离不开金融体系的支撑。

🏛️

银行的职能

银行是金融体系中最重要的组成部分之一，也是公众接触最频繁的正规金融机构。银行承担存款保管、转账结算、贷款支持等基础功能，同时在保障资金安全、提升支付效率、服务实体经济方面发挥重要作用。

🔄

资金如何流动

在日常场景中，资金通常通过存款、支付、贷款、投资等方式实现流动。个人和企业将闲置资金存入银行，银行再通过信贷等方式支持消费、经营与发展，形成有序的资金循环。

📚

为何需要金融服务

金融服务帮助个人和企业更有效地管理资金、应对不确定性、实现跨期安排。了解金融与银行的基本概念，有助于公众更理性地看待金融服务，更有效地识别风险，并作出更稳健的判断。

金融的社会意义

金融不仅关系个人财富管理，也与社会经济运行息息相关。有效运转的金融体系能够将社会闲散资金引导至最需要的领域，推动生产效率的提升与经济结构的优化。

金融素养的起点

了解金融与银行的基本概念，是提升金融素养的第一步。理解金融工具的本质与边界，有助于在面对不同产品和信息时保持清醒判断，避免因认知偏差产生不必要的风险。

MODULE 02

常见金融产品

不同金融产品对应不同功能和使用场景。理解常见金融产品的基础特点，是理性使用金融服务的重要前提。

🏦

存款

存款是最基础、最常见的金融产品，主要作用是保障资金安全、满足流动性需求，并获得一定利息收益。对于风险偏好较低或短期有资金安排需求的用户而言，存款通常是较为稳健的资金管理方式。

流动性好 本金安全 利息固定
📄

贷款

贷款是银行向个人或企业提供资金支持的重要业务，常见于住房、消费、经营等场景。在使用贷款时需重点关注贷款利率、还款期限、月度负担和自身偿还能力，避免因过度借贷带来长期财务压力。

住房贷款 消费贷款 经营贷款
💳

银行卡与支付工具

银行卡和支付工具主要用于存取款、转账、消费和结算，是现代金融服务中最常见的基础工具。随着数字金融发展，支付更加便捷，但便捷并不意味着可忽视安全——规范使用账户、保护个人信息同样十分重要。

借记卡 信用卡 移动支付
📈

理财产品与基金

理财产品和基金通常用于财富管理，适合有一定资金安排需求并愿意承担相应风险的用户。与普通存款相比，这类产品在收益表现上存在更多变化，也伴随一定波动和不确定性。选择时应重点了解产品属性、风险等级及信息披露内容。

存在波动 需了解风险

保险的基础作用

保险的核心功能是风险保障，其主要价值在于帮助个人和家庭应对疾病、意外等不确定事件。保险并不是单纯追求收益的工具，而是家庭风险管理的重要组成部分。正确认识保险的保障属性，有助于建立更完整的金融风险防护意识，避免将保险单纯视为投资渠道。

⚠️
选择金融产品前，请注意

不同产品对应不同功能，既不能简单以"收益高低"进行判断，也不能脱离自身需求盲目选择。在购买任何金融产品前，务必了解其风险属性、适用人群和期限安排，根据自身实际情况审慎决策。

MODULE 03

风险与收益

风险与收益是理解金融产品时必须把握的基本关系。树立"收益与风险相匹配"的意识，是理性投资和风险识别的重要前提。

⚖️

风险与收益的基本关系

通常情况下，收益预期越高，所伴随的不确定性也往往越大。这是金融市场中的基本规律。对于普通用户而言，理解这一关系，有助于在面对各类产品宣传时保持清醒判断，避免被"高收益"噱头所误导。

🔍

常见风险类型

在金融活动中，风险并不只意味着"亏损"。风险还包括资金流动受限（流动性风险）、市场价格波动（市场风险）、信用违约（信用风险）等多种可能性。不同产品面临的风险类型各不相同，了解风险来源才能客观判断其是否适合自己。

👤

风险承受能力

风险承受能力是个人金融决策的重要参考。它通常与收入水平、家庭负担、资金用途、年龄阶段以及个人对波动的接受程度有关。例如，短期内需要使用的资金，通常更适合配置在安全性和流动性较高的产品中，而不应盲目追求高收益。

🗂️

分散风险的基本思路

分散风险是金融管理中的重要原则。将全部资金集中于单一产品、单一机构或单一渠道，容易放大单一事件带来的影响。根据自身目标合理配置资金，有助于提升整体稳健性，降低非必要风险。

🚨
重要风险提示

金融市场中不存在长期稳定的"高收益、零风险、稳赚不赔"。面对此类宣传，应保持高度警惕，增强风险识别能力，坚持通过正规渠道了解产品信息，避免因片面追求收益而忽视潜在风险。本页面内容仅作金融知识科普，不构成投资建议。

⚠️ 风险意识教育

牢固树立风险防范意识

风险管理是金融素养的核心组成部分。理解风险类型、评估自身承受能力、坚持分散配置，是稳健参与金融活动的基本原则。

📉

市场风险

因市场价格波动导致资产价值变化的风险。股票、基金、理财产品均可能受到市场环境变化的影响，价格可能上涨也可能下跌。

💧

流动性风险

资金被锁定、无法在需要时灵活取出的风险。部分产品设有固定期限或赎回限制，购买前需充分考虑自身的资金流动需求。

📝

信用风险

交易对手无法履行还款或偿债义务的风险。选择金融产品时，应了解发行主体的信用状况，并优先通过持牌正规机构开展业务。

MODULE 04

信用与征信

信用是现代金融活动的重要基础，也是个人长期金融行为的综合体现。良好的信用记录，是参与规范金融活动的重要基础。

01 什么是信用

信用反映的是一个人是否能够按照约定履行责任、按时还款、诚实守约。在银行业务和日常金融生活中，良好的信用记录具有重要价值，不仅影响贷款审批和额度，也体现个人的金融契约精神与责任意识。

02 什么是征信

征信是对个人信用行为的客观记录。金融机构在开展贷款、信用卡等业务时，通常会结合征信信息，对申请人的履约情况和信用状况进行综合评估。征信并不是单纯的"好"或"坏"，而是帮助金融机构了解用户历史行为的重要依据。

哪些行为会影响征信

一些常见行为可能对个人征信产生影响，例如：贷款或信用卡逾期未还、长期拖欠欠款、频繁申请信贷产品等。这些情况可能影响后续金融服务的获得，也提醒我们在使用金融工具时要更加重视责任与规范，养成按时还款的良好习惯。

⭐

良好信用的价值

良好的信用记录不仅有助于顺利办理贷款、信用卡等金融业务，也可能在利率、额度等方面获得更有利的条件。同时，良好信用体现了个人的金融素养与契约意识，是参与规范金融活动的重要基础。

🔐

如何维护信用记录

维护良好信用，首先要做到按时还款、合理借贷、量力而行。主动关注账单情况和还款安排，避免因疏忽形成不良记录。同时，妥善保护个人身份信息与账户信息，避免因他人冒用带来信用风险。

MODULE 05

金融诈骗防范

提高风险识别能力，增强安全防范意识，是每一位金融消费者都应具备的基本素养。懂金融，不仅要了解产品，更要学会识别风险、保护资金安全。

📞

冒充客服诈骗

冒充银行、监管机构或电商平台客服，以账户异常、风险处置等为由，诱导用户提供验证码或转账至"安全账户"。

💰

虚假理财诈骗

以超高收益为噱头，通过网络平台或社交媒体推销虚假金融产品，前期可能允许小额提现以建立信任，后续诱导大额投入后卷款跑路。

🔄

刷单返利诈骗

以"在家兼职""刷单返利"为名，要求垫付资金完成任务，初期返还小额佣金诱导信任，随后以各种理由要求追加资金，最终消失。

🏢

非法集资诈骗

以高额回报为承诺，通过非正规渠道向公众募集资金，通常不具备合法金融业务资质，资金安全无法保障，极易导致重大损失。

⚖️

冒充公检法诈骗

冒充公安、检察院、法院等国家机关人员，以涉嫌违法为由施压，要求受害人将资金转入所谓"监管账户"配合调查。

🔗

钓鱼链接诈骗

通过短信、社交媒体发送伪装成官方网站的钓鱼链接，诱导用户输入账号、密码及验证码，从而盗取账户资金和个人信息。

高风险信号识别

遇到以下情形，请立即提高警惕，不轻信、不转账、不透露个人信息：

!要求立即转账或限时操作
!索取短信验证码或支付密码
!发送不明来源链接或二维码
!诱导下载陌生"专用"软件
!承诺明显不合理的高额收益
!要求转账至"安全账户"

账户与信息安全保护

1

保护敏感信息

银行卡号、密码、验证码、身份证号等信息不应随意泄露，也不应通过不明渠道提交。

2

使用正规渠道

办理金融业务时，应优先通过银行官网、官方App、正规营业网点等渠道获取服务和信息。

3

遇疑立即核实

如遇可疑情况，应立即停止操作，通过银行官方客服或营业网点进行核实确认。

4

异常交易及时处置

若已发生异常交易，应尽快联系银行采取止付、冻结等措施，争取将损失降到最低。

🔴
正规金融机构郑重提示

正规金融机构不会通过电话或短信要求客户提供支付密码、短信验证码，也不会要求向所谓"安全账户"转账。接到此类要求，请立即挂断，并拨打银行官方客服电话核实。增强风险意识、坚持正规渠道，是防范金融诈骗的有效方式。

MODULE 06

理性投资与金融素养

理性投资强调长期思维、稳健判断和独立分析，而非盲目追逐热点或因短期波动作出冲动选择。对于普通用户而言，理性投资本身就是金融素养的重要体现。

01

原则 什么是理性投资

理性投资是指在充分了解自身需求和风险承受能力的基础上，审慎作出金融决策。它强调基于事实和逻辑作出判断，而不是受短期情绪、市场热点或他人建议左右，从而在复杂的金融环境中保持稳健。

02

原则 先规划，再决策

在进行任何金融决策前，首先应明确资金用途、使用期限和流动性需求。不同阶段的资金安排，应匹配不同的产品类型和风险水平。先做好规划，再进行选择，才能避免"先购买、后理解"的被动局面。

03

原则 不盲从、不冲动

面对市场信息和产品宣传，保持独立思考尤为重要。热门产品、朋友推荐或网络信息，不能替代对产品本身的判断。了解产品性质、阅读关键信息、关注风险提示，是作出理性选择的重要步骤。尤其面对高收益承诺时，更应坚持审慎原则。

04

原则 长期主义与稳健意识

稳健的财富管理更强调长期积累和合理配置，而不是频繁交易和短期投机。对于零基础用户而言，提高金融素养并不意味着掌握复杂理论，而是逐步形成正确的风险观念、理性的资金管理习惯和规范的金融行为方式。

提升金融素养的方法

金融素养的积累是一个持续的过程。从以下几个方向出发，可以有效提升个人金融认知水平。

📰

持续学习基础知识

通过银行、监管机构、权威媒体等渠道持续获取金融基础知识，建立系统性的金融认知框架，避免碎片化信息带来的误判。

🏛️

关注权威信息来源

优先参考银行官方发布内容、金融监管机构公告及权威财经资讯，对来源不明或夸大宣传的信息保持审慎态度。

✅

坚持正规渠道办理

通过持牌金融机构的官方渠道开展金融业务，拒绝通过非正规平台、陌生人介绍或不明链接办理任何涉及资金的操作。
